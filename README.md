# Tina Yeager Wellness Coach

Your AI wellness coach, powered by Claude and trained on the Sustainable Weight Loss course.

## Files
- `public/index.html` — the chat interface your students see
- `api/chat.js` — the secure backend (keeps your API key safe)
- `vercel.json` — deployment configuration

## Deployment Steps (follow with Claude's guidance)

1. Create GitHub account at github.com
2. Create new repository, upload these files
3. Create Vercel account at vercel.com
4. Connect Vercel to your GitHub repository
5. Add environment variable: ANTHROPIC_API_KEY = your key
6. Deploy — Vercel gives you a live URL

## Your API Key
Get your Anthropic API key at: console.anthropic.com
